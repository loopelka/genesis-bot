ЗАДАЧА: Реализовать полноценную систему корзины и оформления заказа для Genesis Peptide Store.

ВАЖНО:

Проект уже работает в продакшене.
Запрещено ломать существующий функционал.

Существующий механизм быстрого заказа через кнопку “Заказать” должен остаться полностью рабочим.

Корзина должна быть дополнительным способом оформления заказа.

=========================================
ЦЕЛЬ

После внедрения пользователь должен иметь возможность:

1. Просматривать каталог как сейчас.
2. Добавлять товары в корзину.
3. Менять количество товаров.
4. Удалять товары.
5. Видеть общую сумму заказа.
6. Оформлять заказ из нескольких товаров сразу.
7. Отправлять администратору один итоговый заказ.

=========================================
ТРЕБОВАНИЯ К UX

Карточка товара должна содержать две кнопки:

🛒 Заказать сейчас
➕ В корзину

Текущая кнопка заказа не удаляется.

=========================================
КОРЗИНА

Пользователь должен иметь доступ к корзине:

* из главного меню
* после добавления товара
* из каталога

Функции корзины:

* просмотр содержимого
* увеличение количества
* уменьшение количества
* удаление позиции
* очистка корзины
* итоговая сумма
* оформление заказа

Если корзина пустая:

Показать сообщение:

“Корзина пуста”

и кнопку:

📋 Перейти в каталог

=========================================
ХРАНЕНИЕ

Использовать отдельный сервис.

Создать:

services/cart_service.py

Создать:

carts.json

Корзина должна сохраняться между:

* перезапусками бота
* рестартами контейнера
* деплоем новой версии

Запрещено хранить корзину только в MemoryStorage.

=========================================
ВАЖНО ПО ДАННЫМ ТОВАРА

В carts.json НЕ хранить:

* название
* цену
* остаток
* дозировку

Хранить только:

{
“product_id”: 12,
“qty”: 2
}

Все данные товара получать из products_service в момент отображения корзины и оформления заказа.

Это необходимо для актуальности цен и остатков.

=========================================
ОФОРМЛЕНИЕ ЗАКАЗА

FSM:

waiting_name
waiting_contact
waiting_country
waiting_comment
confirming

=========================================
ШАГ КОНТАКТА

Пользователь должен иметь выбор:

1. Ввести телефон.
2. Ввести Telegram username.
3. Ввести любой другой контакт.
4. Нажать кнопку:

“Использовать мой Telegram”

Если username существует:

Связь: @username

Если username отсутствует:

Связь: Telegram ID 

Телефон не обязателен.

Не валидировать формат контакта.
