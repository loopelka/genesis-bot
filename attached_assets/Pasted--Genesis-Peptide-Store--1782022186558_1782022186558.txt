Проведи полный аудит каталога Genesis Peptide Store.

ВАЖНО:

Не создавать новые категории.

Финальные категории должны остаться ТОЛЬКО такими:

* 🔥 Контроль веса
* 💪 Рост мышц и спортивные результаты
* 🩹 Регенерация и восстановление
* ✨ Эстетика и кожа
* 🧠 Когнитивное здоровье
* ❤️ Долголетие и Wellbeing
* 📦 Расходники

Категории:

* 📚 Каталог
* 📚 Полный каталог

должны быть полностью удалены из проекта.

ЗАДАЧА №1

Проверить products.xlsx построчно.

Для каждой записи проверить:

* название
* дозировку
* цену
* наличие
* категорию

Сверить данные, которые реально используются ботом, с данными из products.xlsx.

Если найдены расхождения — исправить автоматически.

ЗАДАЧА №2

Найти все товары, которые сейчас находятся в категории:

📚 Каталог

После завершения работы в этой категории не должно остаться ни одной позиции.

ЗАДАЧА №3

Распределить все товары из категории “Каталог” по существующим категориям согласно их основному назначению и механизму действия.

Использовать медицинскую и фармакологическую логику.

* MT-2 (Melanotan 2)
* GLOW
* KLOW
* NAD+
* PT-141
* CJC-1295 без DAC
* CJC-1295 с DAC
* CJC-1295 + IPA
* AOD9604
* Oxytocin
* 5-amino-1MQ
* Adipotide
* Ara-290
* KPV
* VIP
* Cerebrolysin
* HCG
* Botulinum toxin

Каждый товар должен оказаться в одной из существующих категорий.

Не оставлять товары без категории.

ЗАДАЧА №4

После перераспределения проверить:

Общее количество строк товаров в products.xlsx

и

Общее количество отображаемых товаров в каталоге бота.

Количество должно совпадать.

Не должно быть потерянных товаров.

При несовпадении исправить.

ЗАДАЧА №6

После завершения сформировать отчёт:

1. Какие товары были перенесены между категориями.
2. Какие ошибки были найдены.
3. Какие дозировки были исправлены.
4. Какие цены были исправлены.
5. Сколько товаров стало в каждой категории.
6. Подтвердить, что категория “Каталог” полностью удалена.
7. Подтвердить, что кнопка “Полный каталог” полностью удалена.
